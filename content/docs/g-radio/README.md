## G|Radio
喜歡聼廣播？G|Radio 可以讓您收聽G|TV直播！

無需安裝，使用地址： https://ourhimalayas.github.io/g-radio/

**備注**
* 此應用為非官方發佈的軟件
* 以廣播形式收聽直播節目，如果您的網絡沒問題，請直接使用 [G|TV 官方網站](https://gtv.org)

### 使用方式

<img src="how-to-guide.png" alt="使用方式"/>

* 在刷新列表或者播放電臺時候，如果遇到網絡故障，請重複刷新或者點擊電臺重試。
