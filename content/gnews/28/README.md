## 加拿大保守党支持率激增 特鲁多赢得紧急大选的机会减少
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1527420/)

栏目记者/责任编辑：文真



![](https://assets.gnews.org/wp-content/uploads/2021/08/Gnews-Logo-29.png)





在过去的几个月，特鲁多借由新冠疫情，在加拿大大力推行疫苗接种和疫苗护照政策，引起加拿大民众批评，加拿大多地爆发反疫苗护照游行。

自2019年9月的选举以来，特鲁多领导的自由党在议会中只掌握了少数席位，使他不得不依赖对手政党的支持才能通过立法。特鲁多近期召集的紧急联邦选举，动机在于赢得议会多数席位，并将新冠大流行期间的各种手段转变为未来四年的强制政策。
![](https://assets.gnews.org/wp-content/uploads/2021/09/3-26.jpg)图片来源： 网络
此次大选的早期投票开始后，民调显示，反对党保守党的支持率为33%，而特鲁多的自由党为31%，左倾的新民主党为19%。保守党支持率高于自由党，使特鲁多赢得大选的机会减少。

在周四晚上的辩论中，特鲁多受到了来自各方的抨击。保守党领袖奥图尔批评了特鲁多在应对气候变化方面的记录和他触发投票的动机，以及应对中共的外交策略，指责特鲁多有 “巨大的野心”。此前，特鲁多已经历了此次选举的重大失利。保守党已经在魁北克省赢得了决定性支持。
