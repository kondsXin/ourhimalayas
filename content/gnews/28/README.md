## 新冠疫苗损害法案进入二读辩论 已接种疫苗议员忧心忡忡

栏目记者/责任编辑：安吉拉



![](https://assets.gnews.org/wp-content/uploads/2021/08/Gnews-Logo-29.png)





新冠疫苗损害法案进入二读辩论 已接种疫苗议员忧心忡忡，英国下议院于上周五9月10日就新冠疫苗损害法案进行二读辩论，克里斯托弗·乔普爵士在开篇对是否可以在限定时间内公正的完成该条例草案。乔普爵士表示，包括自己，已接种两针新冠疫苗在座议员，应庆幸目前暂时没有较为严重不良反应。本法案是关于在对抗新冠疫情中，接种疫苗遭受伤害甚至死亡的不幸者，政府更不应该在提供赔偿计划方面玩花样。

根据9月9日政府黄卡最新数据显示，有435份关于重大血栓和低水平血小板的报告，包括74人死亡；报告还显示已经有767例心肌炎，这种情况在医学历史上闻所未闻。其他还包括月三万五千份月经紊乱报告和各种影响。乔普爵士微颤地说道，报告称有1632份涉及接种后不久就发生了死亡。
![](https://assets.gnews.org/wp-content/uploads/2021/09/1-68.jpg)图片来源：网络
英格兰公共卫生局一直在监测接种新冠疫苗后入院和死亡的病人数据，包括今年接种后1到3个月发生的死亡人数报告，却没有按规定格式提供。

在下议院辩论中，就民众发起的疫苗致伤害赔偿请愿书，副议长萨利·按·哈特（Sally-Ann Hart）以政府政策无需讨论为借口刻意回避。

按照1979年法案，受害者不得不面对两个难题：首先，申报需达到60%残疾率的门槛；第二，最高赔付只有12万英镑。乔普爵士认为，针对目前新冠疫苗造成的大面积伤害，政府应进行由法官领导的专项调查，而不是压制信息和验尸官报告。那些接种新冠疫苗遭受严重伤残的人，必须得到支持和赔偿。按照英国议会流程，该草案必须经过所有阶段才能成为法律文书，最早要在2022年夏天才会被列入法规汇编。
