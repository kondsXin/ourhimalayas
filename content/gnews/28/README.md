## 蒙古：接种新冠疫苗后反成为疫情最严重国家
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1523978/)

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



9月9日，一则关于蒙古的新冠病毒的病例曲线图表引起震惊。图表分析显示，蒙古已经超过以色列，成为世界上新冠病毒 病例占比最高的国家。在最近一波疫情开始之前，他们已经为全国超过 60% 的人口接种了疫苗。更加讽刺的是，在大规模疫苗接种之前，蒙古根本就没有冠状病毒。

![](https://assets.gnews.org/wp-content/uploads/2021/09/0910-morning-6-11.004.jpeg)

人们对此表示疑惑：蒙古为什么不动声色的这么早打新冠疫苗，更有人直接怒斥：这是灭族！

2021年2月初，蒙古批准了俄罗斯的卫星五号疫苗，中共疫苗紧跟其后；5月8日，中共国药控股集团通过世界卫生组织的利益链条，获得紧急授权。截至至5月底，蒙古分别向俄中两方购买了一百万剂和四百万剂新冠疫苗，并对全国超过三分之四的人口接种了第一剂疫苗。

蒙古在经济上依赖中共，“疫苗政治和经济”下，蒙古可以出口更多的煤炭给中国，也被要求进一步放开边境。蒙古也获得了部分辉瑞、印度产阿斯利康疫苗，但人口接种的大部分新冠疫苗来自中国。为鼓励国药疫苗的使用，蒙古政府向每位接种中共疫苗的公民提供五万图格里克，约113元人民币的奖励。

新冠病毒和疫苗是中共的生物武器，新冠疫苗引发巨大的人类灾难还只是刚刚开始。
