## 樱花深度报道┃习近平正在祸害中国经济（下）
` 东京樱花团` [轉載自GNews](https://gnews.org/zh-hans/2623627/)

编译：东京樱花团/Hopkins
 
【编者按】目前中国经济正处于危险之中。习近平对经济遭受的双重打击负有不可推卸责任。首先是他的疫情清零政策，这导致了经济衰退，并可能使经济陷入停顿状态。其次，习近平所谓的“新发展理念”，党必须领导一切，一连串的罚款、新法规和清洗导致占国内生产总值 8% 的充满活力的科技行业停滞不前。同时，习近平的意识形态经济对世界具有重大负面影响。尽管刺激措施可能会刺激需求，但可能会出现更多的封锁，从而危及正在陷入衰退的全球经济。在这个世界第二大经济体中，个人独裁的缺陷给中国经济带来灾难是显而易见的。本文编译自《经济学家》2022年5月28日的封面文章“How Xi Jinping is damaging China’s economy”。
 ![](https://assets.gnews.org/wp-content/uploads/2022/05/图片2-17_1653813553.jpg)（图片说明）习近平的新发展理念，（来自央广网） 

```
政府希望正在进行的一项大规模刺激计划将帮助其实现 2022 年 5.5% 的官方增长目标，并在大会召开前平息紧张情绪。 5 月 19 日，总理李克强敦促官员“果断行动”恢复增长，央行下调抵押贷款利率。该党试图安抚惊恐的科技大亨。下一步可能是一项由债券融资的大型政府基础设施计划。
```

```
但更多的债务和大规模的混凝土建设不会抵消严厉封锁的必要性带来的损害，也不会降低习近平经济模式带来的风险。它仅仅涉及扩大经济中生产力最低的部分的范围：政府经营的部分。中国的产业政策曾经取得了巨大的成功，例如在先进电池领域建立了全球主导地位。习近平希望新一批国家投资基金和技术能够让决策更加灵活。但不要忘记所有令人沮丧的失败，从锈带工业（注：指败落的工业园区）到微芯片业。
```

```
与此同时，经济中最具生产力的部分——私营部门——的激励措施遭到破坏。你可以在金融市场上看到这一点，金融市场出现了大量资金外流。资本成本上升：中国股票的交易价格比美国股票低 45%，接近创纪录的差距。投资者和企业家的计算方式正在发生变化。一些人担心，任何企业的财务优势都会被一个怀疑私人财富和权力的政党所限制。风险资本家表示，他们已经转向押注最大的补贴，而不是最好的创意。 40 年来第一次没有主管经济部门进行自由化改革。没有私营企业，增长将受到严重影响。
```

```
习近平的意识形态经济对世界具有重大影响。尽管刺激措施可能会刺激需求，但可能会出现更多的封锁，从而危及正在陷入衰退的全球经济。在商业上，中国的规模和成熟度让跨国公司无法忽视。但据报道，苹果正在采取更多措施来重新平衡中国以外的供应链。中国某些领军企业可能会在 2030 年代主导某些行业，但西方可能会成为对中国产品更加谨慎的进口国。在外交方面，越来越少雄心勃勃和独立的私营部门参与意味着中国在海外的存在将更多地由国家主导和政治化。正如我们关于中国和非洲的特别报告所解释的那样，它可能会变得更加险恶，同时效果也会越来越低。
```

```
个人独裁的危险
```

```
而在一个更加封闭的中国生活又如何呢？虽然人们在网上发泄关于封锁和失业的言论，但由于监视、和对党的目标的垄断性宣传，这不太可能转化为骚乱。一些技术官僚不同意该国的左倾，但缺乏反对它所需的力量和勇气。从精英政治的黑匣子中可以看出，现年68 岁的习近平似乎没有对手。然而，在可能让他至少在 2027 年之前掌权的党代会前夕，在这个世界第二大经济体中，个人独裁的缺陷给中国经济带来灾难是显而易见的。
```
