## 乌克兰议会确认比特币合法化
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1523184/)

栏目记者/责任编辑：安吉娜
![This image has an empty alt attribute; its file name is Bar-3.png](https://assets.gnews.org/wp-content/uploads/2021/07/Bar-3.png)
9月9日乌克兰议会通过一项法律，确定比特币在该国合法化并加以监督，并正式提供相应税收的清晰界限。不同于萨尔瓦多将比特币作为法定货币，乌克兰议会的态度只是正式允许比特币企业在国内运营，为乌克兰在未来尝试加密货币制定了一些准则。

![](https://assets.gnews.org/wp-content/uploads/2021/09/Slide2-5.jpeg)

目前多个国家以行动表态似乎要将比特币推向世界储备货币，背后是否同人民币国际化有关联暂不明晰，但从爆料革命中我们明确得知，比特币涨跌比例和时间全部都受共产党和华尔街大佬操控，最终是洗钱、贩毒的工具。
