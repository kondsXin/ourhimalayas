## 读《道德经》第六章
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1539689/)

作者：香草山健身部 – 书记己嗝屁！
![This image has an empty alt attribute; its file name is Bar-3.png](https://assets.gnews.org/wp-content/uploads/2021/07/Bar-3.png)
【**原文**】
谷神不死，是谓玄牝。玄牝之门，是谓天地之根。
绵绵呵！其若存！用之不堇。

【**译文**】
生养天地万物的道（谷神）是永恒长存的，这叫做玄妙的母性。玄妙母体的生育之产门，这就是天地的根本。连绵不绝啊！它就是这样不断的永存，作用是无穷无尽的。

![](https://assets.gnews.org/wp-content/uploads/2021/09/战友心声.001.jpeg)图片来源：www.terra-sun.net

**谨出作者理解，释如下：**
这里“谷神”指什么呢？有把他比喻成山谷之谷，虚空的意思。谷神就是空虚之神，是万物产生的总根源。道能生养天地与万物，又没有形体，神妙难识，所以老子称道为谷神。谷神就是指生养天地万物的道。所以谷神如果从字面上理解就是养育之神。

其次，“玄牝”又是什么含义？玄是指高深幽远。牝，指母体。泛指具有生养功能的雌性或女性。玄牝就是指自然界最大最最的母体。自然界最大或最高的母体只有能生养天地和万物的母体了。而老子认为，道生养天地和万物，是天地和万物之母。所以，“玄牝”也是道的另一个称谓。

“谷神不死”，是说道永恒存在。“玄牝之门，是谓天地根”，是说永恒存在的道好比伟大的母性，是养育天地万物的根本。

“绵绵若存，用之不勤”，讲道的特性。这句在老子的定义中，道是养育天地万物的母亲，但是这位母亲却以虚和无的形式存在。它生养着天地和万物，但万物却见不到它形体；它支持万物，使万物生生不息。

老子在这一章里继续说明“道”的特征。他所运用的方法仍是比喻、借代。他用“谷”象征“道”，说明“道”既是空虚的又是实在的；他用“神”比喻“道”，说明“道”生万物，绵延不断；他用“玄牝之门”比喻“道”是产生万事万物根源，等等。他想说明“道”的作用是无穷无尽的，从时间而言，它历久不衰，天长地久。从空间而言，它无处不在、无穷无尽。它孕育着宇宙万物而生生不息。

本章用简洁的文字描写形而上的实存的“道”，即继续阐述第四章“道”在天地之先的思想，用“谷”来象征“道”体的虚状；用“神”来比喻“道”生万物的绵延不绝，认为“道”是在无限的空间支配万物发展变化的力量，是具有一定物质规律性的统一体。它空虚幽深，因应无穷，永远不会枯渴，永远不会停止运行。这种支配万物发展变化的力量，就是对立统一规律。

“谷神不死”，体现出“道”的永恒性，即恒“道”。“玄牝之门”是产生万事万物的地方，它的作用非常之大。“玄牝之门”、“天地根”，都用来说明“道”为产生天地万物的始源。把神秘莫名的“道”喻之为母性动物的生殖器官，是非常贴切地描述了无所不能的，生育万物的“道”的特性。
