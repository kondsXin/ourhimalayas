9月8日，美国本福德号（USS Benfold）伯克级导弹驱逐舰驶入美斯奇夫礁12海里内海域，直接挑战了中共宣布本月1日生效的《海上交通安全法》。该法要求外国籍船只进入“中国领海”，应当向海事管理局报告。如果不报告，中共将按照有关法律法规和规章规定进行处理。美国第七舰队在一份声明中说，依照联合国海洋法公约，像美斯奇夫礁这样的低潮高地在高潮时淹没在水面下，因此不能产生领海。美斯奇夫礁上的填海造地、设施的建设不改变这一规则的适用。因此美舰实施了“正常行动”。

据“香港自由新闻”周六（11日）报道，谷歌已经响应去年7月至12月期间香港政府提出的三项要求，向港府提供了用户数据，这使其成为自去年6月香港国安法颁布后，第一个公开表示其遵守地方当局要求提供用户数据的美国科技巨头。去年，谷歌与其它美国科技和社交媒体巨头一起宣布，除非通过美国司法部同意，否则香港当局对任何用户信息的请求将不会得到满足。现在该公司的立场已发生了逆转。

英国太阳报指出，最新研究称染疫恐让肾脏衰老30年，且有约百万人受害。染疫后遗症通常是疲倦和呼吸问题，但美国“退伍军人健康管理局”研究员发现，感染新冠病毒会让肾脏衰老30年。他们在“美国肾病学会期刊”刊登研究指出，专家分析了8万9000名康复退伍军人和160万名康复民众，发现有35%肾脏可能受损或失去功能。该研究资深作者艾阿里表示，染疫后的前6个月，罹患肾脏病风险比未染疫者高35%，且肾功能下降30%的风险高了25%，肾功能下降40%的风险高出44%，下降50%的风险则高62%。参与研究的病患中，至少220人出现肾功能丧失85%现象。
