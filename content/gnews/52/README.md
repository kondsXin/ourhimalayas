#### 撰稿人：东京樱花团/希望
![](https://assets.gnews.org/wp-content/uploads/2021/09/b5bfafae051b4dcb-1.png)
我的孩子没有上幼儿园，都是在家里我们自己抽空带他读读书，我和妻子忙时，他就自己玩。虽没有去幼儿园，可他学到的知识却是其他孩子不能比的。从儒家经典到佛经5岁时孩子都已经熟读了。今年终于上小学了，虽未上幼儿园，但孩子一点也不羞涩，反而融入环境很快。自从知道了文贵先生，就被他大丈夫气魄所吸引，被他那为正义为众生的大胸怀所折服。看盖特看直播已经成了我的一种生活方式。要知道一个人的生活方式是长时间习惯累积的结果，一般很难改变。而在全世界范围内影响了一群数量庞大的人改变了自己的生活方式，可见郭文贵先生的人格魅力！

孩子见我经常看直播看盖特，有一次就问我：“爸爸，这个爷爷是谁？你怎么经常听他说？他能打过魔王吗？”我说：“这是郭爷爷，很厉害奥，妖怪见了他吓得尿裤子。”孩子又问：“钱上面的那个魔王也能打过吗？”我说：“可以！不过那个魔王已经死了，现在郭爷爷要领着大家把他的魔子魔孙给铲除。”

有一天他放学回来跟她妈说：“共产党是魔鬼，是坏蛋！”我笑着把他叫过来说：“你们学校是不是给你们讲共产党好了？”他点头。我说：“以后除了爸爸妈妈不能在同学和老师跟前说。你要保护妈妈和爸爸，魔鬼不喜欢别人说他坏话。”孩子点头好像明白了什么。我摸了摸他的头悄悄地说：“爸爸给你点赞。你说的对！但，我们打妖怪时也要保护好自己。”

由此可见，经过爆一代的洗礼，共匪想对爆二代进行洗脑几乎是不可能的。这些爆二代随着年龄的增长、阅历的丰富和新中国联邦精神的影响，那么他们对善恶的判断力，对是非美丑的鉴别力，对社会责任的担当及面对邪恶的勇气一定是不可估量的。

最近世界各地接连爆出接种疫苗的人都出现不同程度的副作用，各行各业猝死的人也层出不穷。这一现象直接说明了疫苗对人体的伤害是不可逆的。可以说接种过新冠疫苗的人即使未来活下来，他的基因也会有缺陷。基于这一点，爆二代的择偶标准亦会发生根本性的变化。最起码，目前新中国联邦的适龄男女的择偶对象不会选择打过疫苗的。

想一下，别说数以亿计，就是数以百万计的爆二代他们的生存环境、生活方式、价值观和世界观以一种裂变反应的方式影响着未来的世界，那将是多么强大的力量。经过爆一代的耕耘，爆二代的延续传承，世界的政治、经济、文化、制度、教育的格局将会发生巨大变化。

这些未来的爆二代爆三代会在喜体系的环境熏染下追求更高更有品位的生活方式，会以全新的理念改变这个世界。
