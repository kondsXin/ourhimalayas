## 微软跟中共加强合作，公司高管对中共国技术滥用发出警告

![](https://assets.gnews.org/wp-content/uploads/2021/09/O575PLNMNJOCRJGPTCUJ4YU2NI-22.jpg)2017 年微软联合创始人比尔盖茨和中国总理李克强（Thomas Peter/AFP via Getty Images）

埃里克·霍维茨（Eric Horvitz）博士今年在向国会作报告时说，“对于世界上任何一个珍爱个人自由的人来说，中共国在国内对人工智能的使用程度史无前例，让人不寒而栗。” 这份报告是霍维茨在一个小组工作的结果，该小组的任务是就人工智能向国会提供建议。该报告发布之际，微软CEO布拉德·史密斯（Brad Smith）正在与中共高级官员会面，商讨深化双方在在数字技术方面的联系以及微软在中共国的业务。

微软允许中共国对其子公司领英进行严格审查。领英为西方用户创建中、西两个版本的个人资料。当用户发布了中共政府所禁止的词汇时，领英会关闭该用户的中文版信息。3月，领英更进一步，禁止了对一位身居英国的中国评论人士的服务。

报告还强调了中共国使用人工智能更高效地操控人脸识别技术所带来的危险。作者写道，中共国利用人工智能对维吾尔族的镇压，“预示了专制政权将如何使用人工智能为其审查制度、跟踪公民的物理行踪和数字活动、扼杀异议提供便利。”微软为其 Azure 云服务在中共国扩展了数据中心，该云服务包括一个人脸识别软件包。

中共国在3月黑客了微软的数十万个服务器。一些学者认为该攻击是为了收集数据训练其人工智能模型。作为对此攻击的回应，微软扩大了其在中共国的业务。
