据中共国[新华社9月15日消息](http://www.news.cn/fortune/2021-09/15/c_1127862902.htm)称：中共国有“多路资金”正紧盯养老金。公开信息显示，4家银行理财子公司获批开展养老理财产品试点之后，目前已达11只公募养老FOF新品步入发行期。这进一步证实，中共终于通过三年的布局，开始把黑手伸入老百姓的养老钱。

该消息公开信息显示，中共国参与公募养老FOF发行的公司分别是[华夏基金](https://finance.sina.com.cn/money/fund/jjyj/2021-09-06/doc-iktzscyx2647067.shtml?cref=cj)、[兴证全球](http://fund.jrj.com.cn/2021/09/13085933426479.shtml)、[嘉实安康](https://finance.sina.com.cn/money/fund/jjyj/2021-06-10/doc-ikqcfnca0177220.shtml)、[泰达宏利](https://finance.sina.com.cn/money/fund/jjyj/2021-08-10/doc-ikqcfncc1944095.shtml)、[汇添富](https://news.stcn.com/sd/202108/t20210824_3587725.html)、[华商嘉逸](http://www.cet.com.cn/xwsd/2964925.shtml)等。其中，除了南方富誉稳健养老一年FOF在9月15日发行以外，其他6只产品全部已于9月13日发行。其中你会发现不包括新华人寿、平安养老2035(FOF)等大牌保险公司的加入。据所谓业内人士透漏，中共国要把养老金作为未来经济的第三支柱。众所周知，国内工资基数一直在涨，在九月份结束疫情期间个人社保缴费象征性的优惠后突然猛涨到2000多。这是由原来的900多每人到1600每人后突然提升到2000多。[**阅读更多**](https://asia.nikkei.com/Spotlight/Caixin/China-plans-1.72bn-national-pension-company)

感兴趣者通过上述这些所谓新三板投资机构，都是赔本和在赔本底线上徘徊的垃圾公司，如今恒大完了，平安也将要倒，这些垃圾公司却成了新的白手套，协同中共吸老百姓血汗挣来的养老钱，并且中共野心——目标基金是押宝2040年。一个不可能的事，可见，中共习、王最后的疯狂是执行沉船计划了。

综上所述，人们还可以看到中共国在继续加大各种限制后缺乏好的投资环境，有经验的投资管理人才也少，缺乏长期限资金，又无法真正做到风险分散。加上信息披露等于无的前提下。国际机构对中共国公募产品进行评级不足，FOF将会成为真正的风险。 中共国因此让投资者自担风险。而FOF投资又来源于多个机构，做不好风险对冲，则不可能避免市场大幅波动带来的净值过多回撤，因此，这将是导致社保基金血本无归的金融骗局。从金融体系庞大的恒大和平安集团可见，未来中共国又将会哀嚎遍地，血流成河。因为社保基金是打着合法的国家级骗局，有谁可以幸免？[**阅读更多**](http://www.nifd.cn/Uploads/Paper/7936a30d-7c98-4320-9c1d-cba99283b217.pdf)

疫情持续两年，国内贫困人口在持续扩大的同时，很多人一下子就没办法缴纳社保费，这意味着看病就医可能得不到保障，可见王岐山所说 “老百姓吃三年草”的邪恶计划正在实施。并且明确告知：[自2021年9月1日起，职工退休一年内需要进行退休资格认证，否则养老金将自动暂停发放](https://new.qq.com/omn/20210807/20210807A07UCT00.html)。这表明政权破产后，老百姓拿不到退休资格认证。
