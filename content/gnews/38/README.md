## 中共学者人脸识别技术被澳方指责违反道德准则

![](https://assets.gnews.org/wp-content/uploads/2021/09/C9F8F5F3-55F0-4A12-B7F2-6A8864C4826E_cx0_cy10_cw0_w650_r1_s.jpg.jpg)中共学者人脸识别技术被澳方指责违反道德准则 网络图片
据路透社9月15日报道，澳大利亚科廷大学9月15日要求期刊出版商《威立在线图书馆》撤销该校一位前教职员工刘万泉和其他中共国大学学者2018年共同发表的一篇研究论文。该研究帮助中共国人脸识别软件更好地识别少数民族的维吾尔族人，违背了学校的道德守则。

科廷大学表示这项研究是非正式的，由中共国政府资助，并没有得到科廷大学的合约或文件支持，只是提到刘万泉在科廷大学计算机研究系工作。刘万泉和他的共同作者从中共国大连民族大学招募了几百位维吾尔族、藏族和朝鲜族学生，收集并建立了一套人脸形态数据库。

科廷大学要求威立在线图书馆撤销刘万泉论文只是一系列有关中共国政府侵犯新疆维吾尔族以及其他少数民族人权事件中的最新一起，人脸识别技术则是监控和打压维吾尔族人的帮凶。
