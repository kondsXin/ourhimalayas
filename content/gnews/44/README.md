一家全国性报纸的高级官员说：在旋转印刷机市场上占有最大份额的东京机械制作所，不仅与五家全国性报纸有业务往来，而且还与所有地区的地方大报有业务往来。没有东京机械制作所，报业就不会存在，也就是说东京机械制作所是一家极其重要的公司，而这些只有知情者才知道。

东京机械制作所在股市上备受关注。中国银行旗下的投资公司–亚洲发展资本（ADC）已经购买了该公司多达39%的股份，可证券分析师们却装作没看见。虽然东京机械制作所系上市企业, 但在报纸变得更加电子化的当下，我认为它的股价不会有很大的成长空间。所以, 其收购的目的尚不清楚。这次收购导致该公司的股价大起大落，在9月的第二周突然达到高点，然后在接下来的一周达到低点，震惊整个报业。9月10日，40家报纸和新闻机构发表声明，表达他们对此事的关注。”东京机械制作所还承揽了旋转式印刷机的维护工作。任何不可预见的问题都可能使我们停止出版报纸”。如果报纸的根基被外国资金攫取，日本的新闻界可能处于危险之中。

东京机械制作所的律师顾问提问：亚洲发展资本的收购目的是什么？

收购方回答：我们收购股份的理由是基于股票被整体低估和对商业环境的研究。我们不想取代目前的管理层，也不想要求进行任何业务操作上的改变。不会像人们担心的那样，对媒体产生影响。
