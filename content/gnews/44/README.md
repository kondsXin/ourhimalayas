9月16日，乐天董事长兼社长三木谷浩史在个人推特发表对日本移民政策的看法。

针对反对接受移民的传统观点，三木谷先生明确表示不同意。他说：“我理解移民政策的焦虑，但我认为没有别的办法。这已经不是幕府末期攘夷，不接受事实是不行的。”他进一步解释说：“一个人口减少的国家，没法经济发展，科技需要吸引世界一流的人才。少子高龄社会，需要补充劳动力，接受移民。在这个基础上，如何保持日本特有模式是其中要思考。”

但三木谷先生的部分关注者却表达了反对意见，其中一位关注者认为，当前日本面临的主要困境是一流的国际人才不愿来日本，首要的应该是查找问题原因。对于劳力补充，则可通过加大AI技术投资来破解，如果轻率地放开移民政策，则会影响相关科技投资并拉低国民收入水平。寄望于移民政策，不如在解决出生率下降方面着力。

日本老龄化少子化问题由来已久，导致劳动力缺乏问题突出，但由于对维持单一民族的传统观念很强，这就导致日本官方和民间都对于是否放开移民政策感到左右为难、十分纠结。

作为在日多年的华人，笔者认为，不管日本移民政策最终是否完全放开，但至少会进一步松动和灵活，对于人才的引进力度必将加大，毕竟AI技术尚不能完全替代人力。当前，新中国联邦已经通过爆料革命在全世界树立起了良好形象，同时还通过全球农场加大与各国政府和民间的沟通。**作为一名普通的在日战友，必须对自己高定位高要求，不断地提升和完善自己，把自己打造成真正的人才，既展现新中国联邦人形象，更是感恩和回馈日本的实际行动**。
