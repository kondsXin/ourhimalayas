比尔和梅林达·盖茨基金会在一个主要支持中共国有企业的投资基金中购买了一支投资基金超过2000万美元的股票。

该基金中最大控股是国有商业银行和中国工商银行，该银行的现任和前任都是中共高层管理人员。部分国有的中国招商银行也是该基金持股量最大的银行之一。

包括饮料制造商贵州茅台、中国长江电力和华润三九医药股份有限公司在内的国有企业也跻身基金前五大投资之列。

这项投资是在《国家脉搏》揭露了微软创始人与中共国营宣传活动的关系之后进行的，包括那些掩盖新冠病毒起源的宣传活动。盖茨基金会还与中共国官方的科学组织进行了广泛的合作，这些组织与该政权的军队有联系。

他还投资了被美国国防部标记为北京军事代理人的中共国营电力公司。
