**摘要翻译：**前美国太平洋司令部司令哈里斯呼吁华盛顿重新评估其对台湾防务承诺的不明确的长期政策，并警告说中国正在寻求“控制”这个自治岛屿。

华盛顿——前美国太平洋司令部司令哈里斯呼吁华盛顿重新评估其对台湾防务承诺的不明确的长期政策，并警告说中国正在寻求“控制”这个自治岛屿。

美国驻韩国大使哈里斯在最近接受共同社采访时表示，中共国正在寻求“先孤立然后控制台湾”，美方就北京对台北加大军事压力发出警告。
![](https://assets.gnews.org/wp-content/uploads/2021/09/photo_l.jpg)前美国太平洋司令部司令哈里，哈里斯于2021年8月在科罗拉多州接受采访时发表讲话。（共同社）
拜登总统的美国政府要求中共国停止对台湾的军事、外交和经济压力，同时通过军售等方式帮助这个民主岛保持“足够的自卫能力”。

与此同时，美国政府在动用武力应对中共国对台进攻的问题上一直保持着模棱两可的立场，这种策略被称为“战略模糊”。

战略模糊政策是在美台关系于1979年成为“非官方”之后通过的，华盛顿将其外交承认从台北转向北京。

该政策不仅是为了阻止中国对台湾使用武力，而且是为了阻止台湾寻求独立，因为如果发生冲突，北京和台北都无法确定美国是否会介入以保卫台湾。

哈里斯表示，根据《台湾关系法》，美国应该“更明确我们的责任”，这是一项管理与台湾非官方关系的美国立法，并指出华盛顿“并不一致”，尤其是在对台军售问题上.

“如果你在出售台湾或任何其他国家的产品上不一致，那么他们怎么可能——从长远来看，他们如何充分规划他们的军事准备？” 前大使说。

他的言论似乎表明，当时对台湾的武器销售因当时的美中关系而被推迟。

哈里斯的继任者菲利普·戴维森 (Philip Davidson) 在3月份表示，中共国可能会“在未来六年内”尝试入侵台湾。

美国印太司令部现任指挥官约翰·阿奎利诺上将也警告说，“这个问题比大多数人想象的更接近我们”，但没有具体说明他预计中共军队何时有能力入侵该岛。

台湾和中国大陆自1949年因内战分裂以来一直被分开管理。将台湾视为叛徒省份的北京此后一直努力将该岛纳入其阵营。

他抨击北京镇压香港民主、侵犯中国西部新疆地区穆斯林维吾尔少数民族的人权、盗窃知识产权以及对几乎整个南海的“非常规的领土主张”。

“我们相信中共国政府……不信守诺言，”他说。

在应对中共国在该地区日益强硬的态度时，哈里斯表示，鉴于美国和日本拥有 F-35 隐形战斗机和宙斯盾舰等共同防御能力，联合举行军事演习“至关重要”。

他还敦促日本和韩国搁置在战时问题上的争端，专注于印太地区的“今天正在发生的事情”。

他说，美国与其在亚洲最亲密的两个盟友之间的合作“对于成功应对我们三个国家面临的挑战至关重要”，其中包括与中共国有关的挑战。

简评：美国对台湾在政策始于1970年后期，当时是美国政府面临外交承认从台湾到北京的转向阶段。这样背景下的对台模糊政策，与目前中共国对民主台湾岛的虎视眈眈随时进攻野心现状已经不适用了。亚太前司令官哈里斯完全意识到该政策不能适用当前的美台现状，所以站出来强烈建议重新评估美台政策。通过此译文让大家知道美国政府有人看到并意识到当前的台海危机，并希望美国政府接受这一信号及时修订新的美台政策，保护台湾迫在眉睫。
