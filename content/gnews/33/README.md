## 澳大利亚与美英“共命运”对抗中共国
` 英國倫敦喜莊園 Himalaya London Club UK` [轉載自GNews](https://gnews.org/zh-hans/1538141/)

2021年09月17日

![](https://assets.gnews.org/wp-content/uploads/2021/09/merlin_163332447_970d5616-75e0-4fc6-9557-6a0e8e140544-master1050-1.jpg)澳大利亚与美国和英国达成了购买核动力潜艇的协议，以加强本国的柴油动力常规潜艇舰队。 RICHARD WAINWRIGHT/EPA, VIA SHUTTERSTOCK

2021年9月17日纽约时报储百亮发表署名文章，在与北京的关系几年来急剧恶化之后，澳大利亚于9月16日宣布了一项新的防务协议，让美国和英国帮助其部署核动力潜艇，这将对澳大利亚军事力量起到重要推进作用。

选择美国核动力潜艇传递了一个明确无误的信息。通过获得重型武器和绝密技术的做法，澳大利亚已决心在未来世代与美国共命运——用莫里森的话说是“永远的伙伴关系”。该协议将为深化两国军事关系开辟道路，提高人们对澳大利亚会加入到与北京的任何军事冲突中来的预期。

这是一个重大的战略赌注，代表了莫里森政府内部对该地区日益恶化的安全环境、中共国军力的增强，以及中共国有意愿用胁迫手法追求国家利益的巨大担忧，但同时也是一个决定性的时刻，即澳大利亚认为美国将在与中共国的大国竞争中获胜，并继续作为太平洋地区的主导力量和稳定力量，尽管代价在增长——中共国最近对澳大利亚强硬路线的报复包括大幅度消减煤炭、葡萄酒、牛肉、龙虾和大麦的进口，拘留了至少两名华裔澳大利亚公民等。

![澳大利亚总理斯科特·莫里森与英国首相鲍里斯·约翰逊、美国总统拜登一起举行新闻发布会。](https://static01.nyt.com/images/2021/09/16/world/16oz-subs-2sub/merlin_194822847_2af57531-edd3-43b6-98c3-10f00f9401c3-master1050.jpg)澳大利亚总理斯科特·莫里森与英国首相鲍里斯·约翰逊、美国总统拜登一起举行新闻发布会。 MICK TSIKAS/EPA, VIA SHUTTERSTOCK

澳大利亚将成为第二个（之前是英国在1958年）获得美国核动力潜艇技术的国家。对美国来说，支持一个亲密的亚太盟友的决定，代表着它应对中共国军事快速增长的具体升级。澳大利亚废除了与法国签署的建造常规潜艇的巨额协议。

美国国务卿布林肯星期四(9月16日)赞扬澳大利亚面对中共国经济报复而不屈服，并表示美国与澳大利亚站在一起。美国、澳大利亚和英国领导人星期三宣布了新的聚焦印太地区的安全合作伙伴关系。一位拜登政府高级官员星期三说：“这项合作伙伴关系不是针对或关于任何一个国家；它是关于推进我们的战略利益，维持基于规则的国际秩序，并推动印太地区的和平与稳定。”

澳英美协议虽然并未谈及北京，但各方第一时间点评几乎都认为新协议的针对目标就是中共国。美国印太盟友纷纷表示欢迎，而中共国方面则谴责“冷战思维”并警告此举可能引发军备竞赛。

众多美英分析人士认为，此次澳英美协议地缘政治意义非常重大，远不止新闻媒体聚焦的“美国50年来首度对外输出核潜艇技术”，以及澳大利亚可以协助“巡航南海”这么简单。协议是英国和美国全球战略重心转移到印太地区的实际表现。英国首相约翰逊指出，协议意味着未来数十年——几代人的时间，澳英美将展开世界上最复杂、最具挑战性，也是尖端科技含量最高的科技和军工复合体全面整合。他将这一整合过程与60多年前二战之后美英军工科技全面整合相比较。而英国原保守党党魁，英国议会下院重量级议员邓肯·史密斯直接撰文，认为美英澳协议是“自由世界”重新定义与中共国关系的“新时代的起点”。
