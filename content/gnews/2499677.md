摇晃不停的视频捕捉到了在黑暗中领唱的女人，她清澈的嗓音为爱国歌曲注入了一种精致的美，掩盖了她周围的荒凉背景。
 
因为她在马里乌波尔破碎的、地狱般的防空洞里唱歌，坐在她旁边的合唱队低声伴唱。

```
她名叫卡特琳娜（Kateryna）。她去年完成音乐学业后参军，正好21 岁。作为乌克兰战士英雄乐队的一员，她在被围困的工厂中做最后一搏，为生命而战。
```
 
上周四，另一名被困战士尼克·马克（Nik Mark）发布了卡特琳娜手持枪唱两首歌的镜头——她在全国范围内迅速被誉为钢铁鸟，因为她的位置和军事绰号是“鸟”。
 
昨天，在最后一批平民从工厂撤离后，被包围的乌克兰军队领导人举行了在线新闻发布会，宣布他们将与弗拉基米尔·普京的入侵部队战斗到死。
 
亚速团副指挥官斯维亚托斯拉夫·帕拉马尔上尉说，只要我们还活着，我们就会继续战斗，以击退俄罗斯占领者。
 
可能有多达 2,000 名乌克兰士兵仍躲在该工厂。他们的抵抗束缚了俄罗斯军队，挫败了普京在今天胜利日阅兵前夺取这座城市的企图。
 
据了解，经过 74 天的战争，亚速海的战略要地港口遭到轰炸摧毁，他们的处境十分恶劣。
 
据信，在可追溯到 1930 年代的大型苏联工厂下方，有数十名受重伤的人与成堆的尸体一起。这些战斗人员的坚决防御受到乌克兰总统泽连斯基的称赞，但他们正遭受食品、药品和弹药严重短缺的困扰。

```
卡特琳娜的歌声使她成为民族反抗的象征，据了解，她来自该国西部的一个小村庄索斯尼夫卡，在那里她是一名富有创造力的学生，以热爱艺术和积极向上的精神而闻名。
```
 
她写歌，在舞台上表演，并在报纸上发表诗歌。从乌克兰西部城市捷尔诺波尔一所艺术学院的歌唱系毕业后，她在参军前在基辅的一家摩托车店工作。
 
“我们经常谈论舞台、艺术和日常生活” 卡特琳娜的一位老师博丹·马尤克说。
 
“她离开音乐去了另一个让她灵魂丰满的职业。她本可以化妆和约会，但选择了机关枪和迷彩装。卡特琳娜的精神非常强大——她的血液中没有一丝恐惧。”
 
其他人说，她一直对她国家的历史和人权着迷，在基辅生活时参加了集会。

```
“她曾经在一次集会上遭到毒打，但她从不放弃”她的朋友瓦莱里娅·帕纳修克回忆道。
```
 
去年春天，卡特琳娜告诉她的朋友，她计划参加乌克兰东部顿巴斯地区与俄罗斯支持的分离主义分子的战斗。
 
上完医学课程后，为了避免惊动父母，她告诉他们她在医院工作——但在女儿到达前线后，他们很快就发现了真相。
 
然而，即使是她最亲密的朋友也没有意识到她困境的严重性，直到上周出现的视频显示她参与了战争中最血腥的战斗之一。
 
“我感到震惊。我无法想象她竟然在马里乌波尔” 帕纳修克说。基辅政府表示，在周末营救了 300 多名妇女、儿童和老人后，所有平民都已从工厂的掩体网络中撤离。
 
从那时起，俄罗斯军队继续试图袭击该处所。征服马里乌波尔将使莫斯科在 2014 年非法吞并的克里米亚和顿巴斯地区之间架起一座陆桥。

```
但被困士兵的家属正在请求国际援助。尤利娅·费多休克的丈夫阿尔塞尼是受伤的乌克兰战士之一，他的腿被一枚炮弹碎片刺穿，
他说工厂里有 600 名士兵受重伤，其中一些人遭受了痛苦的原始式截肢。
```
 
“每天有一两个战士死去。人体腐烂，并发展为败血症。而且根本没有医疗援助——情况是灾难性的。是的，他们有护理人员，但许多人已经死亡。此外，没有什么手术设施。”
 
费多休克补充说，部队正在喝钢铁厂的脏水。“食物只有谷物和面条”她说。
 
来自亚速团的伊利亚·萨莫连科在线新闻发布会上说，他们必须战斗到最后一刻，因为逃到乌克兰控制的土地几乎没有希望，而且他们担心如果被俘他们会被俄罗斯军队杀死。
 
“投降对我们来说是不可接受的。我们不能把这样的礼物送给俄罗斯人，”大胡子军官说。
 
亚速团是在 2014 年入侵发生后成立的志愿民兵组织，目的是与俄罗斯人作战，一直被其一些极右翼创始人的形象所困扰，普京在为他的袭击寻找替罪羊时抓住了这一点。
 
但它已成为乌克兰国民警卫队中最久经沙场的部队之一，两名亚速斯塔尔捍卫者声称他们在马里乌波尔的部队杀死了 2,500 名俄罗斯士兵，伤兵超过 5,000 人，并摧毁了至少 60 辆坦克。
