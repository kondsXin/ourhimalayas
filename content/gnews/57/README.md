## 中共对美团等10家平台企业开展联合行政指导
` 儒为` [轉載自GNews](https://gnews.org/zh-hans/1525929/)

作者：意大利达芬奇农场 儒为
![](https://assets.gnews.org/wp-content/uploads/2021/09/美团.jpeg)封面来源：网络图片
2021年9月10日上午，人力资源社会保障部会同交通运输部、市场监管总局、全国总工会召开平台企业行政指导会，就维护新就业形态劳动者劳动保障权益对美团、饿了么、滴滴、达达、闪送、货拉拉、满帮、到家集团、阿里巴巴、腾讯等10家头部平台企业开展联合行政指导。会议指出，当前新就业形态劳动者劳动权益保障问题较为突出，平台企业要切实提高**政治站位**，准确理解和把握《关于维护新就业形态劳动者劳动保障权益的指导意见》（以下简称《意见》）的精神实质和主要内容，全面贯彻落实好《意见》要求，依法合规用工。

**浅析：**

**中共开始着手对这些平台企业进行接管和整理工作，该查的查、该撤的撤、该抓的抓、该换的换，通过一系列整改把这些平台都变的“又红又坚”；通过一系列指导把这些平台控制在自己治下；通过一系列强调和要求，把这些头部平台企业按照党的要求“走前头做示范”的作用。这真应了美团创始人王兴要将“共同富裕”建立在美团的基因里，美团的使命是“一起更好”，这次真是一语成谶。这些平台包括“吃、穿、用、行”生活中各个方面，掌握了这些平台，就掌握了所有百姓。**

**中共开展联合行政指导，就是合法抢劫的具体表现形式。中共国企业想要做大做强，必须寻找一个“靠山”，这是决定你能走多远、飞多高的基础，这也决定了在这样的环境里是不容许真正有能力的企业家生存和成长。也就是如七哥所说，中共国没有企业家，只有各种“白手套”、只有各种骗子商人、只有各种投机分子。这就是世界邪恶力量在背后操控中共内各个山头压榨百姓赚取利益的原动力，因为只有中共这种一党专政独裁体制才是最好的土壤。**

**美团的背后有红杉资本、泛大西洋资本、华润、银联国际、阿里巴巴、腾讯等的影子；饿了么背后有阿里巴巴、蚂蚁金服、中信产业基金、华人文化产业基金、红杉资本等世界顶级企业和投资机构；滴滴的背后有柳传志；达达和京东的关系；闪送和鼎晖投资的关系；货拉拉、满帮、到家集团、阿里、腾讯，稍微了解些皮毛都感到不可思议，背后始终是那么十几家，二十几家的大公司和投机机构在转来转去。这其中有多少勾兑和交易是我们不清楚的，但我们唯一清楚的就是中共内部斗争的升级，中共内部经济危机在升级。**
