## 秘翻在线：又一家中共国房企危矣 标普下调“花样年”评级至负面

撰稿：青衣
![](https://assets.gnews.org/wp-content/uploads/2021/09/图片1-51.png)
图片来自网络

继中共国房地产龙头企业恒大集团深陷财务危机，濒临破产之际，另一家知名房地产开发商——花样年控股集团有限公司目前也面临同样危机。

据《大纪元》9月15日报道，国际评级机构标准普尔9月14日将中共国房地产开发商——花样年控股集团有限公司（花样年）的展望由“稳定”降至“负面”。在此之前，国际评级机构惠誉和穆迪也已将“花样年”展望下调至“负面”。

根据香港《信报》9月15日的消息显示，标普确认“花样年”的信用评级及已发行美元优先债券的长期发行评级均为“B”。标普解释称，“花样年”降低成本和资产出售计划使其业务所需资源被移走，预计公司营运的稳定性将受考验，应对庞大离岸到期债券也将面临不确定性。

“花样年”在未来几年将有大量债券到期。如果“花样年”不能及时做出行动，其流动性不足以应对到期债务，届时可能还会被进一步下调评级。

近来，“花样年”频频被媒体关注。一方面的原因是，该公司发行的债券被花旗及瑞信的私人银行部门停止接受作为抵押品，即相关客户不能以“花样年”的债券取得抵押贷款。花旗和瑞信银行对“花样年”的票据均给予“0”贷款价值。

另外一个原因是，“花样年”为中共国知名建筑商，它是由前中共国家副主席曾庆红的侄女曾宝宝于1996年创办，并于2009年上市，主要在深圳、成都等地开发房地产项目。曾宝宝一直是执行董事和大股东。

有评论人士认为，“花样年”的危机显示曾庆红在中共国已经失去了权势。如果曾庆红的影响还在，曾宝宝在中共国贷款会很容易。

在标普下调“花样年”评级至负面之前，另外两家国际评级机构也下调了对“花样年”的评级。今年7月16日，穆迪将“花样年”的评级展望从“稳定”下调为“负面”；同时，穆迪还确认了花样年控股“B2”企业家族评级以及“B3”高级无抵押评级。

穆迪副总裁兼高级分析师Celine Yang对此表示，鉴于“花样年”控股严重依赖海外债务融资，该公司将在信贷紧缩和海外资本市场动荡的情况下，面临更高的再融资不确定性。

惠誉在7月7日也同样将“花样年”的评级展望从“稳定”调整至“负面”，并确认其长期外币发行人违约评级为B+。惠誉同时还确认“花样年”的高级无抵押评级；未偿付债券评级为B+；回收率评级为RR4。

根据“乐居财经”6月22日的消息显示，花样年12只存续美元债的总发行金额为41.2亿美元，平均票息为10.71%。其中，票息最高的一笔达15%，为2018年12月发行的3亿美元票据，半年后即将到期；而发行利率最低的一笔也接近7%。平均到期收益率达15.373%。

目前，“花样年”未偿还的海外存续债金额总计40亿美元，平均到期年限为1.52年，其中有4笔将于年内到期，未偿还金额为11.5亿美元。

此外，截至6月22日，“花样年”还有5只人民币存续债券，发行额总计65.73亿元，平均票面利率7.7%。其中，除去将于半年后到期的“18花样年”已偿还5000万，存续债券余额为65.22亿元。

《华尔街日报》9月14日的消息显示，“花样年”控股和广州富力地产股份有限公司多只2023年和2024年到期的美元债券已跌至面值的60%以下，推动其中大部分债券的收益率升破40%。

目前，在中共国经济断崖式下滑，金融市场即将崩盘的情况下，可以预见，房地产企业将不可避免地发生集体“雪崩”，而不仅仅不只是恒大和“花样年”。
