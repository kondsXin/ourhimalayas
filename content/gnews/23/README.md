9月6日，北约秘书长史托腾伯格 于北约军备控制议题会议上示警，中共在没有任何限制或约束下，正扩建大量导弹发射井，迅速发展核武器。他敦促中方应负起全球大国的责任，响应国际限制核武扩增的作法，包含增加相关资讯透明度等。

史托腾柏格在会议上也列出限制核武的优先事项，并表示除了针对俄罗斯，未来须将更多国家纳入核武限制谈判中，且应在限制条约中加入更多种类的武器，包括AI人工智慧等新颖技术。此前8月时有智库报告根据卫星影像，表明中共正在增建数百座导弹发射井，显然正在快速扩张核武数量。主管美国核武的美军将领也警告，中国很快就会超越俄罗斯，成为美国的头号核武威胁。
