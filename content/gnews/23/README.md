9月6日，BBC新闻报道，巴西总统博尔-索纳罗表示，巴西需要通过改变法规，以打击新闻媒体平台“任意删除”个人资料的行为。为此，巴西将推出新的立法，用以保护公民的言论自由。

总统办公室表示，新法令旨在澄清社交媒体公司在其平台上监管内容的权力范围。到目前为止，此项新发布的法令中的细节指出，社交媒体公司在删除帐户或内容之前，必须提供“正当的理由和动机”。
