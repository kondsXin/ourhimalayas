2021年9月12日郭先生的直播中爆料了中共政府官员陈元一桩惊天的贪腐丑闻。作为中共政府跟沙特阿美的谈判代表，陈元从中共与沙特阿美的石油合同中收取回扣1200亿美元，并在他的亲信张宏伟的协助下，将这笔非法的巨款通过巴基斯坦和东方石油公司、以及新加坡等，把财富藏匿到比特币中。陈元利用不具合法性的公权力，为自己谋取巨额利益，已经构成盗国犯罪。

代表中共政府的陈元，通过国家开发银行（国有企业）支付给沙特阿美的交易款项来盘剥自中国百姓，而陈元串通沙特政府及企业为自己谋利，这样的丑陋行为既欺骗了中共组织，又欺骗了民众。纵观中共的历史，类似陈元的犯罪并不是个例，而是普遍存在的现象。中共篡夺的公权力只为红色家族的私利服务。

极权主义属性的中共体制是滋生腐败的温床。试想一下，如果国家开发银行是私有银行，而陈元是私人企业的谈判代表，这中间就不会存在权力寻租的机会。中共始终滥用权力，它们垄断了中国所有的资源，私人企业没有任何机会参与国家级的交易。张宏伟作为这次交易的参与者，他的公司也只是作为帮助中共红色家族洗钱藏钱的工具。

中共体制的官员一贯以不劳而获来贪占别人的劳动成果。依据郭文贵先生的爆料，这桩交易使用的其实是郭先生家族基金起草的合同，因此收取交易佣金的应该是郭先生的家族基金，而不该是中共政府的谈判代表陈元。
