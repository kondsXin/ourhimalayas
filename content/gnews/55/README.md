拜登总统将发布美国历史上最全面的一套联邦疫苗强制措施，影响到成千上万的雇主和数千万美国人。

“拜登将授权所有雇主（100名工人或以上）要求COVID19疫苗接种或每周对病毒进行测试，影响多达1亿美国人”，公共广播公司报道。

据拜登的一位高级行政官员说，不遵守新的病毒政策的企业将面临“巨额罚款，每次违规最高可达14,000美元”。

“此外，拜登将要求参加医疗保险和医疗补助计划的医院和其他医疗机构的医护人员接种疫苗，影响到1700多万工人。”

NBC新闻报道了关于即将到来的联邦疫苗任务的更多细节，它说这些任务“旨在加大对未接种疫苗者的压力，对联邦雇员、大型雇主和卫生保健提供者提出额外要求，政府说这将影响大约1亿工人，超过美国劳动力的三分之二”。
