[**1、清政府122年前信件曝光 否认“西沙群岛”为其领土**](https://www.aboluowang.com/2021/0909/1644379.html)

英国学者海顿（Bill Hayton）在英国国家档案馆发现的一封122年前的英文翻译信中，当时的中国清朝政府的外交部门指称西沙群岛属于公海，并非中国领土的一部分。这个证据显然与当今中共官方声称“南海诸岛自古以来属于中国”的说法有冲突。分析认为，这可能为越南声索西沙群岛主权提供一些有力证据。海顿告诉自由亚洲电台，这封信是清廷“总理各国事务衙门”(相当于外交部)通知英国官方的译本，信中谈到了发生在1890年末的西沙群岛的“贝罗纳案”。“贝罗纳案”指的是德国船只贝罗纳号在西沙群岛周边失事，其运输的铜货被中国渔民偷走。中国政府在信中以这些岛屿位于“公海”、不是中国领土为由“拒绝赔偿”。

[**2、吴亦凡供出47人 名单惊人全是大咖再泄案外案**](https://www.aboluowang.com/2021/0909/1644400.html)

中共为整肃娱乐圈，祭出许多限令，不只“限籍令”、“劣迹艺人”被一一点名，8月时就传出涉嫌犯下9大罪的吴亦凡希望减少刑期，供出47人名单，当时只爆出名单内有赵薇，如今被公开更多人，不少是中国演艺圈赫赫有名的人物，看来中国演艺圈的动荡一时半刻不会结束。网传吴亦凡供出“47星册”后，5日微博又疯传一张截图，疑似是圈内知情人士的对话流出，内容写著，“今天我上面的公司解约无数，刚刚得到消息，第三批，今天立案的杜海涛、张一山、张艺兴、姚晨、杨紫、关晓彤、岳云鹏、迪丽热巴、肖战、李易峰、王一博、蔡徐坤，吴京、霍思燕、佟丽娅待查。”

[**3、中共整治金融“黑嘴” 已查封1793个社群帐号**](https://www.aboluowang.com/2021/0909/1644440.html)

中共网信办日前宣布整治“唱衰唱空”市场的金融”黑嘴”后，9月8日发布首波成果，共处置2929个违规帐号，其中1793个遭封禁，包含3个拥有百万粉丝的帐号。在整治的过程中，中国网路社群平台如微信、百度、抖音、快手纷纷宣布配合官方政策，并定期发布整治报告。此外，上海证监局今日也发布整治财经黑嘴的初步“成果”，总共下架音档、影片、节目3187个，关闭了53个直播间。上海证监局称，严格禁止未经资格认证的个人和机构，发布涉及投资理财等财经领域的内容；加强受理对涉非法证券期货活动网路资讯的举报，对于核实为违规的内容进行调查，提高举报处理效率。

[**4、四川泸州现蓝白分界“阴阳天” 这是要变天吗？**](https://www.aboluowang.com/2021/0909/1644487.html)

9月7日，四川泸州城区上空出现罕见“阴阳天”，半边是蓝天，半边是白云，分界整齐分明。网友惊呼：这是要变天吗？也有网友表示：真正的平分秋色。据当地媒体报导，9月7日，大面积云层出现在四川泸州城区上空，一边蓝天，一边白云。

[**5、大陆有小学要求学生每周宣誓效忠党**](https://www.aboluowang.com/2021/0909/1644564.html)

9月6日，大陆民众王女士向大纪元记者披露，新学期开学后，有学生家长老师微信群中转发一则通知，每周一次学生向中共邪党宣誓表忠心。每周一7点半组织学生宣誓，内容除了所谓“请党放心，强国有我”之外，还要求学生宣誓，“听党话、跟党走，时刻准备报效祖国。”湖北民众吴先生对大纪元表示，他估计学校的这个通知应该是全国性的一个部署。“我路过当地一所小学的时候也听到了类似的‘宣誓’。在中共政权面临内外危机的背景下，他们很害怕年轻一代接触到自由思想进而反抗中共，因此加强对儿童洗脑教育，让他们从小浸泡在谎言里。”他强调，“这是中共通过洗脑儿童，进而把学生家庭同中共存亡捆绑起来的做法。”

[**6、郑州数百名旅游大巴司机在省交通厅前请愿**](https://www.aboluowang.com/2021/0909/1644300.html)

9月6日，河南郑州市数百名旅游大巴司机在省交通厅前请愿，打出“我们要生存、我们要吃饭”的横幅，要求政府保证他们的正常营运，要求能够养家糊口。据大陆官方消息，7月26日14时，由郑州发往新疆的大巴车（车牌号豫AX5006，核载63人，实载63人）行驶到青兰高速平凉泾川境内（距泾川服务区500米处）发生侧翻。事故造成13人死亡，45受伤。

参与请愿的一位司机李先生向大纪元记者透露，他们到省交通厅请愿的导火索就是因为上述事件。他表示，事件引发郑州所有进行省外旅游客运的一千多辆巴士全部停摆，“我们的车不让跑了，就是因为7月26日郑州有一辆旅游大巴车在甘肃那边出事。中央调查组下来调查了，然后提到两证不符。说我们造假。”郑州自7月以来，经历了洪灾、疫情，旅游业已受到重创，此次停运更让司机们的生活雪上加霜。“我们作为最底层真的承受不起。哪怕让我们车动起来，能养家糊口，也不会造成这样的心理负担。”

[**7、湖北石化高材生遭非法关押 公安不给任何说法**](https://www.aboluowang.com/2021/0909/1644298.html)

裴红，约五十岁，是炼油化工专业的高材生，当年考上了研究生，因考虑老家农村经济困难，想早点上班，放弃了读研。1996年，他大学毕业后被分到中石化荆门石化分公司工作。2021年6月30日，荆门市掇刀区派出所警察从他的借调单位——荆门市盈德气体公司，直接将裴红绑架，并非法抄家，抢劫他的电脑、打印机等私人物品，裴红遭陷害，家里被洗劫一空。裴红大约失联半个多月后，单位不见他上班。他家人因长时间联系不上他，从外地赶到荆门市到处找他，甚至到派出所报警。之后，渐渐查到，他被非法关入荆门市看守所。二十多年来，即使在长期的一线倒班和石化炼油的污染环境下，裴红因为坚持修炼了法轮功，精力充沛，脏活、苦活抢着干，深受同事和领导的好评。去年他的家乡发生洪灾，他回家探亲后，慷慨解囊，捐款一万多元救灾。村民们非常感动。
