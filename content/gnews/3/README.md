## 辉瑞公司故意隐瞒疫苗副作用已触犯法律
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1525516/)

翻译：喜国公民
视频来自网络
通常一个制药公司在推出新药之前的至少6个月就要准备好网站内容，有时甚至是提前一年。当新药得到食品和药物管理局批准的那一天就是制药公司推出新药网站的那一天，目的是要让所有人都知道新药得到了批准。

奇怪的是，辉瑞公司并没有为它的新冠疫苗作宣传。原因是，辉瑞不希望作为市场推销者被指控。如果他们推出一个网站，就会有一连串的对他们诉讼和指控。目前是疾病控制和預防中心，美国国家卫生院，食品药物管理局和主流媒体在向人们推销新冠疫苗。这些人一直在误导人民。

但辉瑞作为一个制造商，要为故意伤害人民的行为负责。因为辉瑞公司故意隐瞒了疫苗产生的严重副作用和死亡人数，故意误导食品药物管理局和百姓。

另外，辉瑞需要对公众澄清疫苗的成分以及新药研究申请。 而第三阶段实验并不在卫生保健提供者的概况介绍中。这意味着误导和故意伤害。
