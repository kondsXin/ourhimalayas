过去10个月，中共政府针对电商、网约车、校外辅导到社交媒体等多个行业展开一系列监管，投资者已成惊弓之鸟。暂停新游戏审批预示着中共监管政策的升级，将对开发商业务的长期增长造成打击。从叫停蚂蚁集团，到对美团和滴滴全球展开调查，北京遏制互联网巨头的行动已接近第11个月。

翻手为云覆手为雨，中共随心所欲玩转股市，投资中共股市，血本无归的危险性很高。
