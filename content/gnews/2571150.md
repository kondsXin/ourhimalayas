## 樱花深度报道┃即将到来的大饥荒（上）
` 东京樱花团` [轉載自GNews](https://gnews.org/zh-hans/2571150/)

编译：东京樱花团/Hopkins

```
【编者按】《经济学家》（The Economist）2022年5月21日发表封面文章“The coming food catastrophe”向全球发出警讯：俄罗斯入侵乌克兰战争正在将一个脆弱的世界推向大规模粮食危机，未来几个月将出现可能持续数年的大饥荒，如果战争持续下去，俄罗斯和乌克兰的供应受限，那么可能会有数亿人陷入贫困。政治动荡将蔓延，儿童将发育迟缓，人们将挨饿。世界领导人应将饥饿视为迫切需要全球解决方案的全球性问题。同时解决这个问题是每个人的事，现在请去做一些事情来帮助防止或延缓大饥荒的发生。
```
 ![](https://assets.gnews.org/wp-content/uploads/2022/05/图片3-17_1653125313.jpg) 

```
通过入侵乌克兰，弗拉基米尔·普京（Vladimir Putin）将摧毁远离战场的人们的生命——其规模甚至可能令他都感到后悔。这场战争正在打击因 covid-19、气候变化和能源冲击而削弱的全球粮食系统。乌克兰的粮食和油籽出口大部分已停止，俄罗斯的出口受到威胁。两国共同供应了 12% 的交易卡路里。小麦价格自年初以来上涨了 53%，5 月 16 日又上涨了 6%，此前印度表示将因令人担忧的热浪而暂停出口。
```

```
被广泛接受的生活成本危机的想法并没有开始抓住未来可能发生的事情的严重性。联合国秘书长安东尼奥·古特雷斯（António Guterres） 5 月 18 日警告说，未来几个月将威胁到可能持续数年的“全球粮食短缺的幽灵”。高昂的主食成本已经使无法确定吃饱的人数增加了 4.4 亿，达到 16 亿。近2.5亿人处于饥荒边缘。如果战争持续下去，俄罗斯和乌克兰的供应受限，那么可能会有数亿人陷入贫困。政治动荡将蔓延，儿童将发育迟缓，人们将挨饿。
```

```
普京先生绝不能以食物为武器，虽然短缺不是战争的必然结果。世界领导人应将饥饿视为迫切需要全球解决方案的全球性问题。
```

```
俄罗斯和乌克兰供应全球 28% 的小麦贸易、29% 的大麦、15% 的玉米和 75% 的葵花籽油。俄罗斯和乌克兰贡献了黎巴嫩和突尼斯进口谷物的一半左右；对于利比亚和埃及来说，这个数字是三分之二。乌克兰的食品出口为 4 亿人提供了卡路里。战争正在破坏这些供应，因为乌克兰已经在其水域布雷以阻止袭击，而俄罗斯正在封锁敖德萨港。
```

```
甚至在入侵之前，世界粮食计划署就警告说 2022 年将是可怕的一年。最大的小麦生产国中国表示，在去年因降雨推迟播种后，今年的收成可能是有史以来最差的。现在，除了世界第二大生产国印度的极端气温之外，从美国的小麦产区到法国的博斯地区，缺雨还威胁到其他粮仓的产量。非洲之角正遭受四年来最严重的干旱。
```

```
这一切都会对穷人产生严重的影响。新兴经济体的家庭将 25% 的预算用于食品，而在撒哈拉以南非洲，这一比例高达 40%。在埃及，面包提供了所有卡路里的 30%。在许多进口国，政府无力支付补贴来增加对穷人的帮助，特别是如果它们还进口能源——另一个动荡的市场。
```

```
危机可能会变得更糟。尽管托运人的成本和风险增加，乌克兰在战前已经运送了去年夏天的大部分作物。但俄罗斯仍在设法出售其谷物。然而，那些没有被战斗破坏的乌克兰筒仓里还装满了玉米和大麦。由于 6 月下旬开始，农民无处存放下一次收成，因此可能会腐烂。而且他们缺乏燃料和劳动力来种植之后的植物。就俄罗斯而言，它通常从欧盟购买的种子和杀虫剂可能缺乏一些供应。
```
